# creaser-cipher-project
This code implements a **Caesar Cipher Encryption and Decryption web tool using HTML, CSS, and JavaScript**
**HTML Structure**
Provides a user interface with a text area for input text, a numeric input for the shift value (0â€“25), Encrypt/Decrypt buttons, and a result display area.

**CSS Styling**
Uses a gradient background and a centered container with rounded corners and shadows to create a clean, modern, and responsive design.

**JavaScript Logic**
The caesarCipher() function shifts each alphabetic character by a given number while preserving case (uppercase/lowercase).

Non-alphabetic characters (spaces, symbols) remain unchanged.

Handles negative shifts for decryption.

**Encryption & Decryption**
Encrypt: Applies a positive shift to the text.

Decrypt: Applies a negative shift to reverse the encryption.
